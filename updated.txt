````
# ⚙️ STEP 1 — ADVANCED PREPROCESSING (MUST UPGRADE)

## Add:

```python
from scipy.signal import savgol_filter
import numpy as np

def preprocess(X):
    X_smooth = savgol_filter(X, 11, 2, axis=1)

    # Standard Normal Variate (SNV)
    X_snv = (X_smooth - X_smooth.mean(axis=1, keepdims=True)) / X_smooth.std(axis=1, keepdims=True)

    # Derivatives
    d1 = np.gradient(X_snv, axis=1)
    d2 = np.gradient(d1, axis=1)

    return np.concatenate([X_snv, d1, d2], axis=1)
````

✔ Removes scattering + baseline drift
✔ Critical for soil spectroscopy

```python
class SpectralEncoder(nn.Module):
    def __init__(self, input_dim):
        super().__init__()
        self.encoder = nn.Sequential(
            nn.Linear(input_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 128)
        )

    def forward(self, x):
        return self.encoder(x)
```

### Contrastive loss:

```python
def contrastive_loss(z1, z2):
    return ((z1 - z2)**2).mean()
```

# ⚙️ STEP 3 — LOCAL SPECTRAL MODELING (CRITICAL)

## Idea:

Find similar spectra and train locally

```python
from sklearn.metrics.pairwise import cosine_similarity

def get_local_subset(X, target_idx, k=200):
    sim = cosine_similarity(X[target_idx].reshape(1,-1), X)
    idx = np.argsort(sim[0])[-k:]
    return X[idx]
```

✔ Mimics spectral library approach
✔ Proven to increase accuracy

# ⚙️ STEP 4 — REPLACE CNN WITH TRANSFORMER

## Why:

* Spectral data = sequence
* Transformers outperform CNNs

```python
import torch.nn as nn

class SpectralTransformer(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.attn = nn.MultiheadAttention(dim, num_heads=4)
        self.fc = nn.Linear(dim, 128)

    def forward(self, x):
        x = x.unsqueeze(1)
        x,_ = self.attn(x, x, x)
        return self.fc(x.mean(dim=1))
```

✔ Captures long-range spectral dependencies
